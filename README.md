# Rust Simple Todos CLI

A tutorial project implementing a simple to-do list via console commands

## Installation
```bash
make init
```

## Usage

### Add todo
```bash
make add todo="My Todo 1"
```

### Show all todos
```bash
make todos
```

## Tests
```bash
make test
```
